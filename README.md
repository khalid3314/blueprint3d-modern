## 🖥️ What this app does

blueprint3d-modern helps you create and view floor plans on your Windows PC. It is a modern TypeScript rewrite of blueprint3d, so it is easier to keep up to date and should feel smoother to use.

Use it to:

- open floor plan projects
- draw rooms and walls
- place doors and windows
- review layout changes on screen
- work with a clean, simple interface

This project fits well for home layouts, room planning, and basic floor design work.

## 🧭 First-time use

When you start blueprint3d-modern for the first time, use this order:

1. Open the app.
2. Create a new floor plan.
3. Add walls to shape the room.
4. Place doors and windows where needed.
5. Add furniture or other items if the app provides them.
6. Save your work often.

If you make a mistake, undo the last step and try again.

3. Try another browser if the app opens in a browser.

If the screen looks blank or broken:

- refresh the page

- reopen the app
- try a different browser
